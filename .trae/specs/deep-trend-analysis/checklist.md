# 深度趋势分析 验收清单 (Checklist)

- [ ] **数据完整性 (Data Consistency)**
    - [ ] `DataLoader` 能正确识别指定周（Mon-Sun）的日期范围。
    - [ ] 即使缺少某一两天的日报数据，流程也不会崩溃（应记录警告并基于现有数据继续分析）。

- [ ] **AI 分析质量 (AI Analysis Quality)**
    - [ ] 生成的趋势分析**不是**简单的"本周热门项目列表"。
    - [ ] "二阶推理" (Second-order reasoning) 生效：能产出连贯的叙事（例如：将"硬件限制"与"去中心化"联系起来）。
    - [ ] 输出的 JSON 严格符合预期的 Schema (`title`, `summary`, `content`, `evidence`)。

- [ ] **视觉呈现 (Visual Presentation)**
    - [ ] 新的"深度趋势"版块与现有的"热门项目"列表在视觉上有明显区分。
    - [ ] 排版优化：长文阅读体验良好（合适的行高、字体）。
    - [ ] 移动端适配：叙事文本在小屏幕上也能流畅阅读。

- [ ] **性能 (Performance)**
    - [ ] 批量加载 7 个 JSON 文件不会显著拖慢生成速度。
    - [ ] LLM 分析调用能在合理时间内完成（或有超时处理机制）。
